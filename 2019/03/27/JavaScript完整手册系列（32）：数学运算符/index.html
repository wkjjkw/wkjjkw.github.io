递增 (++)

递增数字。这是一个一元运算符，如果放在数字之前，则返回递增后的值。

如果放在数字后面，它将返回原始值，然后递增。

递减(–)

和递增运算符相似，不过它递减值。

一元负号(-)

返回操作数的负值

一元正号(+)

如果操作数不是数字，它会尝试将操作数转换数字。如果操作数已经是一个数字，它什么都不做。

快速赋值

常规赋值运算符 = ，对所有算术运算符都有一个快捷方式，该方式允许你组合赋值，将第一个操作数和第二个操作数的结果赋值给第一个操作数。

它们是：

+=：加法赋值
-=：除法赋值
*=：乘法赋值
/=：除法赋值
%=：取余赋值
**=：求幂赋值
