如何获取访问者的ip地址并写入header

发表于 2019-01-20 | 更新于 2019-01-28 | 分类于 web前端

背景

最近的一个项目，新增了一项需求：获取访问者的ip地址，并将ip地址写到header中，后端获取ip地址后写入日志进行记录。

由于js本身不具备获取访问者ip地址的能力，所以必须借助于第三方服务才能达到这个目的。网上查找了相关资料，最终确定使用搜狐的ip接口服务（http://pv.sohu.com/cityjson?ie=utf-8）。
